# 복슬퀀트

## 프로젝트 개요
- **프로젝트명**: 복슬퀀트
- **목적**: 다양한 퀀트 투자 전략의 성과를 분석하고 비교하는 Python 기반 백테스팅 시스템
- **주요 기능**: 일시투자 vs 적립식투자, 모멘텀 투자, 변동성 돌파 전략 분석 등

## 명령어
```bash
# 의존성 설치
pip install -r requirements.txt

# 실행 (현재 미완성)
python main.py
```
