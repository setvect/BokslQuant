# 복슬퀀트 (Boksl Quant)
**[Claude Code](https://www.anthropic.com/claude-code)를 이용해서 만들었음**

다양한 퀀트 투자 전략의 성과를 분석하고 비교하는 Python 기반 백테스팅 시스템입니다.

## 1. 주요 기능

- **일시투자 vs 적립식투자** 성과 비교
- **모멘텀 투자** 전략 분석
- **변동성 돌파** 전략 구현
- **백테스팅 엔진** 및 성과 지표 계산
- **시각화** 차트 생성

## 2. 설치 및 실행

### 2.1. 의존성 설치
```bash
pip install -r requirements.txt
```

### 2.2. 프로그램 실행
```bash
python main.py
```
